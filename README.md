# IdőKocka - Pomodoro + Teendőkezelő

## Funkciók
- 25 perc munka + 5 perc szünet (automatikus váltás)
- Teendők: hozzáadás, törlés
- Lokális adatmentés (SQLite)

## Technológia
- .NET MAUI
- MVVM + CommunityToolkit.Mvvm
- SQLite (sqlite-net-pcl)
- Data binding

## Futtatás
1. Nyissa meg a `.sln` fájlt Visual Studioban
2. Válasszon platformot
3. Futtassa
